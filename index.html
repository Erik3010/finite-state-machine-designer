Finite State Machine Designer

An app that is used to design a Finite State Machine.

Some shortcut that you need to know to use this application:

  • Click to select state or arrow
  • Double Click to create State
  • Drag to move the state position
  • Shift + Drag to add a arrow from current state to another state
  • Delete to delete state or arrow
